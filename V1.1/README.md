# FontExtractor V1.1 Prebuilt

This folder contains prebuilt binaries for direct use.

## Files

- `FontExtractorGUI.exe`  
  Portable executable. Run directly.

- `FontExtractorGUI-1.1.0.msi`  
  Installer package for standard Windows installation.

## Do I Need Python?

No.  
These binaries are self-contained for end users and do not require manual Python/dependency installation.
